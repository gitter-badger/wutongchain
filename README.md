&emsp;&emsp;梧桐链主要提供最基础的区块链服务和基于该链的业务服务。梧桐链支持基于私有云和公有云部署和扩展；支持节点可控授权接入；支持多种加密算法、多种共识算法；支持高性能自主智能合约引擎；提供对区块链系统的治理和运维支持，可对整个网络的运行状态进行实时监控。
